7.1 One-sample location

7.1.1 Sign Test

Suppose we have a sample of \(X_1,X_2,...,X_n\) an arbitary continuous distribution. We use the median \(m\) as denote their location parameter(In normal population, we use mean instead). Therefore, \(P(X_i>m) = 0.5\) and \(P(X_i<m) = 0.5\). We want to test \[ H_0: m =m_0 \] Let \[\begin{align} Y_i &= \begin{cases} 1, X_i-m_0>0 \\ 0, X_i-m_0<0 \end{cases} B &= \sum_{i=1}^n Y_i \end{align}\] From the derivation of \(B\), we can see \(B\sim Bin(n,0.5)\), then \(E(B) = n/2\) and \(Var(B) = n/4\). Let \(b^*\) be the observed value of \(B\). The p-value of testing \(H_0:m=m_0\) is \[ p = \left\{\begin{array}{ll} P\left(B \geq b^{*}\right)=\sum_{b=b^{*}}^{n}\left(\begin{array}{l} n \\ b \end{array}\right)\left(\frac{1}{2}\right)^{n} & \text { if } H_{1}: m>m_{0} \\ P\left(B \leq b^{*}\right)=\sum_{b=0}^{b^{*}}\left(\begin{array}{l} n \\ b \end{array}\right)\left(\frac{1}{2}\right)^{n} & \text { if } H_{1}: m<m_{0} \\ 2 \min \left\{\sum_{b=b^{*}}^{n}\left(\begin{array}{l} n \\ b \end{array}\right)\left(\frac{1}{2}\right)^{n}, \sum_{b=0}^{b^{*}}\left(\begin{array}{l} n \\ b \end{array}\right)\left(\frac{1}{2}\right)^{n}\right\} & \text { if } H_{1}: m \neq m_{0} \end{array}\right. \]

7.1.2 Wilcoxon Signed Rank Test

Suppose we have a sample of \(X_1,X_2,...,X_n\) an arbitary continuous and symmetric distribution with median \(m\). We want to test \[ H_0:m=m_0 \] We let \(Y_i = X_i-m_0\), sort \(|Y_i|\) in ascending order and denote the corresponding rank \(R_i\), let \[\begin{equation} S_i = \begin{cases} 1, Y_i>0,\\ -1, Y_i < 0 \end{cases} \end{equation}\] \[ W = \sum_{i=1}^n R_iS_i \] with \(E(W) = 0\) and \(Var(W) = \frac{n(n+1)(2n+1)}{6}\).
